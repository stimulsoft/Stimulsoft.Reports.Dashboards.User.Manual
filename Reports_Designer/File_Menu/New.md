## New

The **New** section contains the following commands:

* The **Blank Report** command allows you to create a new report and open it in the designer;

* The **Blank Dashboard** command allows you to create a new dashboard and open it in the designer;

* The **Get Data** item commands for creating a new report or dashboard with or without demo data;

* A **Search Bar** enables you to find report and dashboard examples with demo data using a search query;

* A **Report Creation Wizards** is a set of wizards that allow you to configure templates step by step.

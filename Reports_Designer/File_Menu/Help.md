## Help

This menu item contains links to various help resources:
* The Documentation command opens a window with the user guide.

* The Support command provides a link to the [technical support page on the official Stimulsoft website](https://www.stimulsoft.com/en/support);

* The Video command provides a link to [Stimulsoft's YouTube channel](https://www.youtube.com/user/StimulsoftVideos);

* The Sample command provides a link to the [examples page on the official Stimulsoft website](https://www.stimulsoft.com/en/samples);

* The Forum command provides a link to the [Stimulsoft forums](https://forum.stimulsoft.com/);

* The Demo command provides a link to [Stimulsoft's demo page](https://demo.stimulsoft.com/);

* The Training Courses command provides a link to the [training courses page on the Stimulsoft website](https://www.stimulsoft.com/en/training-courses).

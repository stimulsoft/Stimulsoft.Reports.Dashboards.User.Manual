## Items Dialog


In the Items dialog you can create, delete, edit items (values, ​​expressions). This window is invoked when clicking the Editor in the Variables dialog. The picture below shows the Items dialog:


![](../../../../images/topics/Data.Data_Dictionary.Variables.Panel_Request_From_User.Items_Dialog_1.png)


![](../../../../images/img_1.png) Control Panel. This panel contains buttons to control items.

![](../../../../images/img_2.png) In the Toolbox displays a list of created items (values, expressions). Keep in mind that the order of items in the list affects sequence of items in the Items field on the Request from User panel.

![](../../../../images/img_3.png) The properties panel. In this panel the properties of the selected item are displayed. The item has two properties: Key and Value.

![](../../../../images/img_4.png) The panel displays the description of the selected property.


Control Panel

As mentioned above, on this panel (see the picture above) the buttons to control items are placed.


![](../../../../images/topics/Data.Data_Dictionary.Variables.Panel_Request_From_User.Items_Dialog_2.png)


![](../../../../images/img_1.png) The New Value button. Used to create a new type of the value;

![](../../../../images/img_2.png) The New Expression button. Creates a new type of an expression;

![](../../../../images/img_3.png) The Select Columns button. Calls a dialog where you can specify data columns as keys and values;

![](../../../../images/img_4.png) The Remove button. Removes the selected item.

![](../../../../images/img_5.png) The Navigation buttons. Used to move selected item up or down in the toolbox.

![](../../../../images/img_6.png) The Close button. Closes the Items dialog saving changes.

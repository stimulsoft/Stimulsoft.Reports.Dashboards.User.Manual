> **Information**
>
> *.mrt - a report or dashboard template file;
>
> *.mrz - a packed report or dashboard template file;
>
> *.mrx - an encrypted report template or dashboard file;
>
>
> *.mdc - a file of the rendered report;
>
> *.mdz - a packed file of the rendered report;
>
> *.mdx - an encrypted file of the rendered report.

## Labels


Labels are titles of X axis (the axis of the arguments) and Y (the axis values). Labels can take any string value. Any string value is transformed according to the selected format. If the report generator failed to convert a value to the selected format, then a direct string value is output. The picture below shows an example of a chart with arguments of Labels. The Format property is set to N:


![](../../../../../images/topics/Report_Internals.Charts.Axes_Area.Axes.Labels_1.png)


Also, Labels have a number of properties such as:

![](../../../../../images/fly.png) **Angle** - sets an angle of inclination of labels;

![](../../../../../images/fly.png) **Antialiasing** -  sets smooth-edged type of  labels;

![](../../../../../images/fly.png) **Color** - sets the labels color;

![](../../../../../images/fly.png) **Font** - sets the font type of labels;

![](../../../../../images/fly.png) **Format** - changes the label format (numeric, percentage etc);

![](../../../../../images/fly.png) **Placement** - changes the position of showing Labels;

![](../../../../../images/fly.png) **Text before/Text after** - shows a text before/after Labels;

![](../../../../../images/fly.png) **Text Alignment** - used for **Y** axis, aligns Labels;

![](../../../../../images/fly.png) **Width** - changes the width of Label.

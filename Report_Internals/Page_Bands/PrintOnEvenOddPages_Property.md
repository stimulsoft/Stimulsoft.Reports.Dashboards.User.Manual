## PrintOnEvenOddPages Property


The PrintOnEvenOddPages property is used to print headers and footers on even/odd pages, for Page Header bands and Page Footer bands.


![](../../images/topics/Report_Internals.Page_Bands.PrintOnEvenOddPages_Property_1.png)

![](../../images/topics/Report_Internals.Page_Bands.PrintOnEvenOddPages_Property_2.png)


The picture above shows a sample of a report with the PrintOnEvenOddPages property of the Page Header band set to EvenPage.


![](../../images/topics/Report_Internals.Page_Bands.PrintOnEvenOddPages_Property_3.png)

![](../../images/topics/Report_Internals.Page_Bands.PrintOnEvenOddPages_Property_4.png)


The picture above shows a sample of a report with the PrintOnEvenOddPages property of the Page Header band set to OddPage.


Three values are available for this property:

![](../../images/fly.png) Ignore. Bands are printed on all pages;

![](../../images/fly.png) PrintOnEvenPages. Bands are printed on even pages;

![](../../images/fly.png) PrintOnOddPage. Bands are printed on odd pages.

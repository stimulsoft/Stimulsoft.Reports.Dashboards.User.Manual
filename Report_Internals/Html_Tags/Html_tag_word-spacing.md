## HTML &lt;word-spacing&gt; Tag

Using the &lt;word-spacing&gt; tag it is possible to define the space between each words. If the &lt;text-align&gt; tag with the "justify" value is used, then the &lt;word-spacing&gt; tag is ignored. This happens because the interval between words is already specified and a line of a text is aligned by both left and right sides. The example below shows how the &lt;word-spacing&gt; tag works:

Test &lt;word-spacing="2"&gt; Test &lt;/word-spacing&gt;Test

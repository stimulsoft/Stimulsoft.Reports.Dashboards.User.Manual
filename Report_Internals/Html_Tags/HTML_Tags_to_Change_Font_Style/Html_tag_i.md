## HTML &lt;i&gt; Tag

The &lt;i&gt; tag is used to define italic text.  The tag can be used in combination with other tags to change the text style. For example, if you enter the following expression:

Test &lt;i&gt;Test&lt;/i&gt; Test

then after calculation the result appearing in the report will be:

Test Test Test

If a tag is not closed then the formatting is applied to from the opening tag to the end of the text expression.

## HTML <ul> Tag

The <ul> tag inserts an unordered list, which is a block level element consisting of a sequence of items, usually displayed with a bullet on the left margin.


<p>How it works!</p>

<ul>

<li>How</li>

<li>it</li>

<li>works.</li>

</ul>


The result of output:

How it works!

- How
- it
- works.

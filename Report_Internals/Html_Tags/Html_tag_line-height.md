## HTML &lt;line-height&gt; Tag

The &lt;line-height&gt; tag sets the height of the text line. The tag is set as the multiplier for the basic line height. By default the value if the &lt;line-height&gt; tag is 1. The example below shows how this tag works:

Test&lt;line-height="1.5"&gt;&lt;br&gt;&lt;/line-height&gt;Test&lt;line-height="0.7"&gt;&lt;br&gt;&lt;/line-height&gt;Test

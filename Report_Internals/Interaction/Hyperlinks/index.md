## Hyperlinks


Hyperlinks are used in report navigation. Also it is possible to use the Interaction.Bookmark and Interaction.Tag properties for this. Hyperlink is set to Interaction.Hyperlink property. When report rendering, the expression, specified in this property, is set to Interaction.HyperlinkValue property. Setting occurs when report rendering. There are three ways of specifying hyperlinks. It is possible to use one of them.

## Break if Less Than Property

The Break if Less Than property can be any number value from 0 to 100. The value of this property affects where it will generate a new page or column. At the time of the report creation, report generator measures the amount of free space on the page as a percentage. If the entire page is empty, it is equivalent to 100 per cent, if the page is full - 0 percent. It should be considered that by default the Break if Less Than property is set to 0, which means that this option is disabled. Therefore, if the value of this property is 0, the report writer compares the percentage of free space on the page with the specified property value. As a result, the report generator will generate a new page or column, if the free space on the page is less than a predetermined value. The property value is a key value, i.e Break if Less Than property cannot be empty, it must take a value.
